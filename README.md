# Agro_Tools

### Descrption :
This is a simple PHP-based Agriculture & Garden Tools Management System that allows users to browse and purchase agricultural tools online. It includes secure user login, product management (admin only), shopping cart, and checkout functionality. The system is designed with HTML and CSS for an easy and clean user interface.

### Key Features :

- User Registration and Login System
- View Available Agriculture & Garden Tools
- Add Items to Shopping Cart
- Secure Checkout with Customer Details
- Admin Login with Product Management
- Session-Based Cart Functionality
- Simple & Attractive UI with CSS
- Basic Order Placement System

 ### How to Run :

  1. Install XAMPP / WAMP on your system
  2. Start Apache and MySQL server
  3. Download or Clone this project into:
  4. htdocs folder for XAMPP or www folder for WAMP
  5. Open phpMyAdmin and create a new database (e.g., agro_tools)
  6. Import the provided SQL file into the database
  7. Open your browser and go to: http://localhost/Agri_Tools/index.php
  8. Register/Login as a user and explore the shop!
